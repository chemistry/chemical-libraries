# @chemistry/common

[![npm](https://img.shields.io/npm/v/@chemistry/common)](https://www.npmjs.com/package/@chemistry/common)

Common data formats and interfaces for the @chemistry library ecosystem.

## Installation

```bash
npm install @chemistry/common
```

## Features

- JMol and JNMol data format interfaces
- Shared types for atoms, bonds, and molecular structures
- Isomorphic — works in Node.js and browsers

## Usage

```typescript
import { JMol, JNMol } from '@chemistry/common';
```
